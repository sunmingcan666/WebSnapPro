# WebSnapPro
WebSnapPro 是一款基于 Python 和 PyQt5 开发的图形化网站内容下载工具，专门用于抓取和保存网站的代码和资源文件。该工具提供了直观的用户界面和丰富的功能选项，让用户能够轻松下载整个网站或特定页面的内容。
